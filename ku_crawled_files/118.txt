    //Use javascript to avoid weather caching
    $kuj(document).ready(function() {
        if(!$kuj.cookie('myweather')){
            var url = '//webmedia.ku.edu/~http/ssi/2012/widgets/cmslawrenceweather.php';
            var weatherval = '<div class="sectiontitle">Current Weather</div>';
            var date = new Date();
            var minutes = 30;
            date.setTime(date.getTime() + (minutes * 60 * 1000));
            try {
            //cross script issue with IE8 workaround
            if (navigator.appVersion.match(/MSIE/) && window.XDomainRequest) {
                // Use Microsoft XDR
                var xdr = new XDomainRequest();
                xdr.open("get", url);
                xdr.onload = function () {
                    var result = weatherval+xdr.responseText;
                    $kuj.cookie('myweather', result, { expires: date });
                    $kuj('#weatherholder').html(result);
                };
                xdr.send();
            } else if (!navigator.appVersion.match(/MSIE/)) {
                $kuj.ajax({
                    type: 'GET',
                    url: url,
                    success: function (data) {
                        date.setTime(date.getTime() + (minutes * 60 * 1000));
                        $kuj.cookie('myweather', weatherval+data, { expires: date });
                        $kuj('#weatherholder').html($kuj.cookie('myweather'));
                    }
                });
            }
        } catch (e) {}
        }else{
            $kuj('#weatherholder').html($kuj.cookie('myweather'));
        }

    });
</script>

                                <h1 class="title" id="page-title">The Learning Studio</h1>
                    <span property="dc:title" content="The Learning Studio" class="rdf-meta element-hidden"></span>    
    <div class="content clearfix">
        <div class="field field-name-body field-type-text-with-summary field-label-hidden"><div class="field-items"><div class="field-item even" property="content:encoded"><script type="text/javascript">
  $kuj(function(){

<p>The Learning Studio is a partnership of KU Libraries, <a href="http://undergraduatestudies.ku.edu">Undergraduate Studies</a> and <a href="http://technology.ku.edu">KU Information Technology</a>&nbsp;to create a dynamic, student-centered space located on the 3rd and 4th floor of Anschutz Library. It builds on the success Anschutz Library already enjoys as the central hub of undergraduate research and learning at KU. The Studio features technology, an integrated service point, and a variety of study spaces. All of this is brought together with librarians, IT professionals and academic achievement&nbsp;staff, to support students in writing, research, study skills and use of technology.</p>

<p>This welcoming environment offers unique opportunities for students to collaborate with their peers, study and research, and find support for their academic work.</p>

<ul>

<p>The open computer areas on the 3rd and 4th floors of Anschutz house more than 100 workstations (<strong><a href="https://labstats.ku.edu/LabStats/public/public.aspx">check availability</a></strong>). These include multiple scanning workstations (with Adobe Design Premium), specialized workstations with assistive technology and accessible furniture, as well as express print and quick access stations.</p>

</ul>

<h3>Technology check-out</h3>

<p>Come visit us at the 3rd floor circulation desk in Anschutz to inquire about technology available for check-out at the Learning Studio.</p>

<p><strong>Laptops</strong><br />
Laptops are available for use in the library building. These cannot be taken outside of the library. Four laptops are also available for extended, out of building check-out.</p>

<p><strong>iPad minis</strong><br />
Two iPad minis are available to check out for use in or outside the library. If neither is available, you can request to be added to the waiting list.</p>

<p><strong>Headphones and voice recorders</strong><br />
These are also available on a first-come, first-served basis.</p>
<strong>Digital cameras</strong>

<ul>
	<li>Canon Vixia HF M30  HD Camcorder (1)</li>
	<li>Canon PowerShot S90 digital still camera (1)</li>
	<li>Sony Cyber-shot DSC-HX5V digital still camera (1)</li>
</ul>
<strong>Calculators</strong><br />
Graphing Calculators:
<ul>
	<li>TI-83 (3 available)</li>
	<li>TI-86 (1 available)</li>
	<li>TI-89 (3 available)</li>
</ul>
Scientific calculators:

<ul>
	<li>TI-30x (1 available)</li>

	<li>Casio fx-260 (1 available)</li>

Financial Calculators:

<p><strong>Tutoring groups</strong>: Space for formal tutoring groups is located on the 4th floor of the Learning Studio. These spaces are scheduled by Tutoring Services (part of the Academic Achievement and Access Center) for tutoring sessions. Reservations are posted outside each space. When not in use, these are available on a first-come, first serve basis for all Anschutz users. <a href="http://www.tutoring.ku.edu">Learn more or sign up for a tutoring group.</a></p>
