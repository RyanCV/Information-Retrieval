            var date = new Date();
            var minutes = 30;
            date.setTime(date.getTime() + (minutes * 60 * 1000));
            try {
            //cross script issue with IE8 workaround
            if (navigator.appVersion.match(/MSIE/) && window.XDomainRequest) {
                // Use Microsoft XDR
                var xdr = new XDomainRequest();
                xdr.open("get", url);
                xdr.onload = function () {
                    var result = weatherval+xdr.responseText;
                    $kuj.cookie('myweather', result, { expires: date });
                    $kuj('#weatherholder').html(result);
                };
                xdr.send();
            } else if (!navigator.appVersion.match(/MSIE/)) {
                $kuj.ajax({
                    type: 'GET',
                    url: url,
                    success: function (data) {
                        date.setTime(date.getTime() + (minutes * 60 * 1000));
                        $kuj.cookie('myweather', weatherval+data, { expires: date });
                        $kuj('#weatherholder').html($kuj.cookie('myweather'));
                    }
                });
            }
        } catch (e) {}
        }else{
            $kuj('#weatherholder').html($kuj.cookie('myweather'));
        }

    });

        <div class="field field-name-field-topcontent field-type-text-long field-label-hidden"><div class="field-items"><div class="field-item even"><h2>Blackboard Spellcheck feature turned on</h2>

<p>The Spellcheck function in Blackboard was turned on in our system Wednesday, April 20, 2016 and is now available in textboxes within Blackboard course sites. This was done after spending much time consulting with the vendor, testing and applying their fixes for this issue to our system.<br />
&nbsp;<br />
Please note that at this time, the Spellcheck function only works as designed in Firefox or Chrome, where it underlines the misspelled word in red and offers suggestions for correction when right-clicking on the word.&nbsp; However, in Internet Explorer, the functionality only partially works by underlining the misspelled word, but does not offer suggestions for correction.&nbsp; While Blackboard is working to correct this issue, we wanted to get Spellcheck turned back on as soon as possible for campus as we get closer to finals.</p>

<p>While we dont anticipate any issues with this tool or with the overall system now that Spellcheck has been re-enabled in Blackboard, please let us know if you experience any Blackboard performance issues. You can contact the KU IT Educational Technology team at <a href="mailto:blackboardsupport@ku.edu">blackboardsupport@ku.edu</a>&nbsp; or 785-864-2600 during office hours, or the Customer Service Center after 5 p.m. at 785-864-8080 or <a href="mailto:itcsc@ku.edu">itcsc@ku.edu</a> .</p>

      <h3>Blackboard Workshops include:</h3>

<div class="accordion">
<p><strong>Blackboard: Getting Started</strong></p>
</div>

<div class="accordion_content">
<p><span style="font-size: small;">This workshop provides a hands-on introduction to the Blackboard Learn interface and organization features. You will focus on navigation, the course style, and the process of planning, adding, and organizing links on the course menu. </span></p>

<p><span style="font-size: small;">Along the way, you will examine different strategies for organizing the course menu, which is the students access point to tools and content. Finally, you will consider ways you can influence your students impressions of the online learning environment.</span></p>

<p><a href="http://apps.ku.edu/~infotrain/workshops.php?option=details&amp;id=100067" target="_blank">Register for Blackboard: Getting Started</a></p>
</div>

<div class="accordion">
<p><strong>Blackboard: Assignments</strong></p>
</div>

<div class="accordion_content">
<div class="span-14 last">
<p>The assignments tool enables you to create, manage, and distribute activities that assess students in meaningful ways beyond traditional objective tests.</p>

<p>The Grade Center's organization and management capabilities are incorporated into the assignments tool, allowing you to provide feedback and grades efficiently.</p>
</div>

<p><a href="http://apps.ku.edu/~infotrain/workshops.php?option=details&amp;id=100011" target="_blank">Register for Blackboard: Assignments</a></p>
</div>

<div class="accordion">
<p><strong>Blackboard: Discussions</strong></p>
</div>

<div class="accordion_content">
<div class="span-14 last">
<p>Online discussions provide some unique benefits. Because students can take time to ponder before posting ideas, you may see more thoughtful conversations play out.</p>

<p>You can observe as students demonstrate their grasp of the material and correct misconceptions. You can extend your office hours by reaching students more often during the week so learning is continuous.</p>
</div>

<p><a href="http://apps.ku.edu/~infotrain/workshops.php?option=details&amp;id=100072" target="_blank">Register for Blackboard: Discussions</a></p>
</div>

<div class="accordion">
<p><strong>Blackboard: Grade Center</strong></p>
</div>

<div class="accordion_content">
<p><span style="font-size: small;">Review tools in the Blackboard Learn Grade Center and set up columns using the new features.</span></p>

<p><span style="font-size: small;">Create new column: numeric, calculated</span><br /><span style="font-size: small;">Use Smart Views to filter the grade columns</span><br /><span style="font-size: small;">Enter and edit grades</span><br /><span style="font-size: small;">Download and upload the grade data</span><br /><span style="font-size: small;">Create feedback for students</span><br /><span style="font-size: small;">Enter Manual overrides to grades</span></p>

<p><a href="http://apps.ku.edu/~infotrain/workshops.php?option=details&amp;id=100013" target="_blank">Register for Blackboard: Grade Center</a></p>
</div>

<div class="accordion">
<p><strong>Blackboard: Groups</strong></p>
</div>

<div class="accordion_content">
<p>In this workshop, you learn to facilitate collaboration among students by setting up groups. In Blackboard Learn, groups can have their own discussion board, blog, journals, wikis, and live chats. Group members can exchange documents, submit group assignments, send email to other group members, and distribute tasks to keep the group on track.</p>

<p><a href="http://apps.ku.edu/~infotrain/workshops.php?option=details&amp;id=100071" target="_blank">Register for Blackboard: Groups</a></p>
</div>

<div class="accordion">
<p><strong>Blackboard: Tests</strong></p>
</div>

<div class="accordion_content">
<p>In Blackboard Learn, you can use tests to measure student knowledge, gauge progress, and gather information from students.</p>

<p>In this workshop, the main focus is on creating tests and questions, choosing appropriate settings, and adding tests to course areas. Along the way, you will learn some best practices for using tests as teaching tools and encouraging academic honesty.</p>

<p><a href="http://apps.ku.edu/~infotrain/workshops.php?option=details&amp;id=100070" target="_blank">Register for Blackboard: Tests</a></p>
</div>

<div class="accordion">
<p><strong>Blackboard: Blackboard Collaborate</strong></p>
</div>

<div class="accordion_content">
<p>Blackboard Collaborate is a web conferencing platform which instructors can use to schedule synchronous class sessions. Collaborate can be launched directly from your Blackboard course.</p>

<p>Synchronous class sessions are a convenient medium for collaborative, interactive, and mobile learning.</p>

<p><a href="http://apps.ku.edu/~infotrain/workshops.php?option=details&amp;id=100078" target="_blank">Register for Using Blackboad Collaborate</a></p>
</div>

<hr /><p>Please visit the <a href="https://blackboard.ku.edu/workshops" target="_blank">Blackboard Workshops</a> page to view additional descriptions and register for upcoming events.</p>
    </div>
