    //Use javascript to avoid weather caching
    $kuj(document).ready(function() {
        if(!$kuj.cookie('myweather')){
            var url = '//webmedia.ku.edu/~http/ssi/2012/widgets/cmslawrenceweather.php';
            var weatherval = '<div class="sectiontitle">Current Weather</div>';
            var date = new Date();
            var minutes = 30;
            date.setTime(date.getTime() + (minutes * 60 * 1000));
            try {
            //cross script issue with IE8 workaround
            if (navigator.appVersion.match(/MSIE/) && window.XDomainRequest) {
                // Use Microsoft XDR
                var xdr = new XDomainRequest();
                xdr.open("get", url);
                xdr.onload = function () {
                    var result = weatherval+xdr.responseText;
                    $kuj.cookie('myweather', result, { expires: date });
                    $kuj('#weatherholder').html(result);
                };
                xdr.send();
            } else if (!navigator.appVersion.match(/MSIE/)) {
                $kuj.ajax({
                    type: 'GET',
                    url: url,
                    success: function (data) {
                        date.setTime(date.getTime() + (minutes * 60 * 1000));
                        $kuj.cookie('myweather', weatherval+data, { expires: date });
                        $kuj('#weatherholder').html($kuj.cookie('myweather'));
                    }
                });
            }
        } catch (e) {}
        }else{
            $kuj('#weatherholder').html($kuj.cookie('myweather'));
        }

    });
</script>

        <div class="field field-name-field-bottomright field-type-text-long field-label-hidden"><div class="field-items"><div class="field-item even"><h3><a href="/wws" style="letter-spacing: -0.05em;">Wounded Warrior Scholarship</a></h3>

<p><img alt="" src="/sites/gmp.ku.edu/files/images/general/Anthony_Schmiedeler_web.jpg" style="letter-spacing: -0.05em; width: 400px; height: 300px; float: left; margin-top: 0px; margin-bottom: 7px;" /></p>

<p>KU is proud to offer a <strong>Wounded Warrior Scholarship</strong> for disabled military servicemembers, veterans, primary caregivers, and surviving spouses/children studying at KU. Continuing in the Fall semester of 2015, KU will offer scholarships up to $10,000, which are renewable for up to four years. <a href="/wws">Learn more...</a></p>

<h3><a href="https://vimeo.com/135496183">KU Student Veterans Center</a></h3>

<p><img alt="KU Veterans Service Center" src="/sites/gmp.ku.edu/files/docs/SVC-LobbyWeb.jpg" style="width: 400px; height: 400px;" /></p>

<p>The new <a href="https://vimeo.com/135496183"><strong>KU Student Veterans Center (KU SVC)</strong></a> will be opening in early 2017. The new KU SVC will establish a “one-stop shop” for veteran services and addressing the three crucial phases of the student veteran’s university education: (1) transition from the military to the campus (first-year experience); (2) persistence to graduation; and (3) transition from the campus to the civilian workforce. This important center will be a valued addition to the support our Student Veterans currently receive at KU. For more information, or to see <a href="http://www.kuendowment.org/stuvetctr">how you can help</a>, see the <a href="/sites/gmp.ku.edu/files/docs/student-veteran-center-12.15.14.pdf">KU Student Veteran Center</a>. If you wish to make a donation to help fund this important resource for our Veterans, check out<a href="https://www.launchku.org/project/1752"> <strong>LAUNCH KU.</strong></a></p>
