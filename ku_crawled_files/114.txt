    //Use javascript to avoid weather caching
    $kuj(document).ready(function() {
        if(!$kuj.cookie('myweather')){
            var url = '//webmedia.ku.edu/~http/ssi/2012/widgets/cmslawrenceweather.php';
            var weatherval = '<div class="sectiontitle">Current Weather</div>';
            var date = new Date();
            var minutes = 30;
            date.setTime(date.getTime() + (minutes * 60 * 1000));
            try {
            //cross script issue with IE8 workaround
            if (navigator.appVersion.match(/MSIE/) && window.XDomainRequest) {
                // Use Microsoft XDR
                var xdr = new XDomainRequest();
                xdr.open("get", url);
                xdr.onload = function () {
                    var result = weatherval+xdr.responseText;
                    $kuj.cookie('myweather', result, { expires: date });
                    $kuj('#weatherholder').html(result);
                };
                xdr.send();
            } else if (!navigator.appVersion.match(/MSIE/)) {
                $kuj.ajax({
                    type: 'GET',
                    url: url,
                    success: function (data) {
                        date.setTime(date.getTime() + (minutes * 60 * 1000));
                        $kuj.cookie('myweather', weatherval+data, { expires: date });
                        $kuj('#weatherholder').html($kuj.cookie('myweather'));
                    }
                });
            }
        } catch (e) {}
        }else{
            $kuj('#weatherholder').html($kuj.cookie('myweather'));
        }

    });

                                <h1 class="title" id="page-title">KU Parent E-Newsletter</h1>
                    <span property="dc:title" content="KU Parent E-Newsletter" class="rdf-meta element-hidden"></span>    
    <div class="content clearfix">
        <div class="field field-name-body field-type-text-with-summary field-label-hidden"><div class="field-items"><div class="field-item even" property="content:encoded"><p><img alt="" src="/sites/parassociation.drupal.ku.edu/files/docs/RS16434_13041_A_22-scr.jpg" style="width: 378px; height: 252px; float: right;" />The KU Parent E-Newsletter is an announcement-only list designed to send updates on selected campus events, reminders for deadlines and dates, and information about University activities.</p>

<div class="content clearfix">
<div class="field field-name-body field-type-text-with-summary field-label-hidden">
<div class="field-items">
<div class="field-item even">
<p><b>Why join the KU Parent E-Newsletter?</b><br />
The KU Parents E-Newsletter is available for parents or guardians of KU students as an information resource in the form of an e-newsletter. This list will bring information about KU straight to your e-mail box on a regular basis. Additionally, announcements concerning KU programs, services, events or activities may come to you from the greater University community.</p>

<p><b>How do I join?</b><br />
Just fill out the <a href="/join-ku-parent-e-list">form</a>. All we need is your e-mail address and screen name.</p>

<p><b>How do I quit?</b><br />
The list is easy to quit if you find that it is not for you. <a href="/unsubscribe-ku-parent-e-list">Unsubscribe</a> to remove yourself from the KU Parent E-Newsletter.</p>

<p><b>How does it work?</b><br />
Once you join, you will be sent an e-mail explaining all of the details. An announcement-only list will only allow administrators at the University of Kansas to send messages. Although some lists allow for the participants to send messages to the list which will then be sent to everyone, this is not the case with the KU Parent E-List.</p>
